# Event / Template Catalog

This catalog contains practical scenario templates in the current authoring format.

## Structure
- `authoring/`: ready-to-load authoring project templates.
- `runtime/`: matching runtime bundle examples (`runtime_rules`, `runtime_conditions`, `runtime_assets`) for each template.

- `night_music_swamp`

- `weather_rain_ambient`
